# voxiom.io-market-bot
A market monitoring bot for the web game voxiom.io

# Installation
```git clone https://github.com/PixelMelt/voxiom.io-market-bot```

```cd src```

```npm i axios```

```npm i kalmanjs```

# Configuration

edit the start of ```marketmonit.js``` with your discord webhook url

# Running

```node marketmonit.js```
